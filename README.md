# TranquiClean

Professional, detail-oriented house cleaning you can actually count on. One cleaner, one standard — every single time.

## About

Single-page website for TranquiClean, an owner-operated house cleaning service in Jersey City, NJ.

## Setup

1. Clone the repository
2. Open `index.html` in a browser, or deploy to Netlify/Vercel
3. Replace placeholder content (phone, email, calendar, affiliate links)

## Deployment

Static site — deploy to any static hosting provider. No build step required.
